# Code of Conduct

We are committed to a respectful, inclusive community.

## Expectations
- Be respectful and constructive
- Assume good intent
- Focus on the issue, not the person

## Unacceptable Behavior
- Harassment, discrimination, or abuse
- Trolling or derailing discussions
- Sharing private information without consent

## Enforcement
Project maintainers may take action to address violations.

## Reporting
Report issues to the maintainers via GitHub issues (mark as sensitive if needed).
